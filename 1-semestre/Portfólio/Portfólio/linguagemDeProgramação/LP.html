Linguagem de Programação

Uma linguagem de programação é um conjunto padronizado de regras e sintaxe que permite aos programadores escreverem instruções que um computador pode entender e executar. Essas instruções formam um código fonte, que pode ser compilado ou interpretado para criar um programa de computador funcional. As linguagens de programação são usadas para desenvolver software, aplicativos, sistemas operacionais, jogos e muito mais.

Variaveis

Uma variável é um elemento que armazena dados na memória do computador para ser utilizado e manipulado durante a execução de um programa. Ela é nomeada de forma que possa ser referenciada no código.

Operadores

São símbolos especiais que realizam operações em operandos (variáveis ou valores). Eles são essenciais para manipulação de dados e controle do fluxo do programa.
Operadores são fundamentais para a construção de lógica em programação, permitindo a criação de algoritmos eficientes e dinâmicos.

Estruturas de decisão


if/else

Essa estrutura de decisão executa um conjunto de instruções se o teste lógico for verdadeiro, e caso contrario ele ira executar outro conjunto de instruções.



Switch

A estrutura de decisão switch permite que o programa escolha diferentes caminhos com base no valor fornecido para avaliação. Se o valor não corresponder a nenhuma das opções especificadas, o bloco de instruções default será executado.

O switch permite que haja um teste lógico para cada case em sua estrutura, como se fossem diversos "ifs" alinhados.

Estruturas de repetição


While


While funciona com uma estrutura que cria um loop, que executa um bloco de instruções diversas vezes enquanto alguma condição for verdadeira.

Do/while


Parecido com o while, o do/while obriga que a as instruções sejam executadas pelo menos uma vez.


For


Seguindo a mesma lógica do while, com apenas uma diferença, a estrutura do for obriga a criação de uma variavel de controle.

Vetores

Vetores (arrays) são tipos de variáveis que reservam espaços consecutivos na memória para armazenar múltiplos valores do mesmo tipo. Para acessar esses valores, usa-se o nome do vetor junto com um índice numérico.


Matriz


Matrizes são vetores com múltiplas dimensões.
Para acessar os elementos de uma matriz, podemos usar laços for aninhados.


Funções


Funções são conjuntos de instruções que são executados apenas quando são invocados. Elas podem receber parâmetros, que são valores fornecidos durante a chamada da função, e também podem retornar valores para o ponto no programa onde foram chamadas.


Recursividade


A recursividade é um conceito fundamental na programação que envolve a capacidade de uma função chamar a si mesma repetidamente até que uma condição de parada seja alcançada.


Programação Orientada a Objetos


A programação orientada a objetos (POO) é um paradigma de programação que organiza o código em estruturas chamadas de objetos.


Estrutura de Classe


A estrutura de classe é um conceito fundamental da programação orientada a objetos (POO), que define um modelo ou plano para a criação de objetos.


Encapsulamento


Encapsulamento é um dos princípios fundamentais da programação orientada a objetos (POO) e refere-se à ideia de reunir dados (atributos) e os métodos que operam nesses dados em uma única unidade, chamada de objeto. Isso significa que os atributos de uma classe são geralmente declarados como privados, impedindo o acesso direto a eles de fora da classe.

Em Java, os modificadores de acesso são palavras-chave que controlam a visibilidade dos membros (atributos e métodos) de uma classe. Existem três tipos principais de modificadores de acesso em Java:

Public


Protected


Private



Getters e Setters

Getters e setters são métodos especiais utilizados para acessar e modificar os valores dos atributos de uma classe de forma controlada, seguindo os princípios de encapsulamento da programação orientada a objetos.

Método Construtor

Método construtor é um metedo que é executado automaticamente quando uma instância de uma classe é criada. O objetivo principal de um método construtor é inicializar os atributos de um objeto, dessa forma garante que ele esteja em um estado válido e consistente desde o momento de sua criação.

Diagrama UML

A Linguagem de Modelagem Unificada (UML) foi desenvolvida para fornecer uma linguagem de modelagem visual padrão, rica em semântica e sintaxe, para a arquitetura, design e implementação de sistemas de software complexos, abrangendo tanto a estrutura quanto os comportamentos desses sistemas.

Classes

Uma classe é um conceito fundamental na programação orientada a objetos que define um tipo de objeto. Ela funciona como um molde ou modelo que descreve as propriedades (atributos) e comportamentos (métodos) que os objetos desse tipo terão. Em essência, uma classe encapsula dados e as operações que podem ser realizadas sobre esses dados.

Classes estáticas

Uma classe estática é uma classe que não pode ser instanciada, ou seja, você não pode criar objetos a partir dela. Todos os membros da classe, incluindo métodos e atributos, são também estáticos, o que significa que eles pertencem à classe em si e não a instâncias específicas.

Sobrecarga

A sobrecarga de métodos é um recurso da programação orientada a objetos que permite a criação de vários métodos com o mesmo nome, mas com diferentes listas de parâmetros, dentro da mesma classe.

Exceções

Exceções são eventos que ocorrem durante a execução de um programa, sinalizando a ocorrência de condições anormais ou erros.

Herança

Herança é um conceito fundamental na programação orientada a objetos que permite a criação de novas classes baseadas em classes existentes. A classe nova, chamada de subclasse ou classe derivada, herda atributos e métodos da classe existente, chamada de superclasse ou classe base. Isso promove a reutilização de código e facilita a manutenção e a extensão de software.

Super classe

Uma superclasse, também conhecida como classe base ou classe mãe, é uma classe da qual outras classes podem ser derivadas. Em termos de herança na programação orientada a objetos, uma superclasse contém atributos e métodos que são compartilhados por uma ou mais subclasses.

Polimorfismo

Polimorfismo é um dos princípios fundamentais da programação orientada a objetos que permite o uso de uma interface única para múltiplos tipos de dados ou classes.
